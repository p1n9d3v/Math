# Algebraic Properties

## Commutative

- 교환법칙
- `a + b = b + a`

## Associative

- 결합법칙
- `a + (b + c) = (a + b) + c`

## Distributive

- 분배법칙
- `a * (b + c) = ( a * b ) + ( a * c )`

## Identities

- 항등원 : 어떤 값(a)과 연산을 했을 때 결과가 원래의 값과 동일하게 만드는 값($a + e = a$)

## Inverses

- 역원 : 어떤 값(a)과 연산을 했을 때 결과가 항등원이 되게 하는 값($a + (-a) = e$)
